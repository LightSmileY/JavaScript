# HTML表格与表单

### 表格

```html
<table border = "1" width = "100%">
    <thead>
        <tr>
            <th colspan = "4">head</th>
        </tr>
    </thead>

    <tbody>
        <tr>
            <td>Cell 1</td>
            <td>Cell 2</td>
            <td>Cell 3</td>
            <td>Cell 4</td>
        </tr>
    </tbody>

    <tfoot>
        <tr>
            <td colspan = "4">foot</td>
        </tr>
    </tfoot>
</table>
```
