# IDE

### 前端编辑器

- ##### WebStorm —— 集成开发环境，功能强大，但体积庞大

- ##### Sublime —— 轻便，速度最快，最性感的编辑器，适合前端，缺点是调试困难

- ##### VS Code —— 目前最热门编辑器，可调试，适合C、Python、JavaScript 等

- ##### Atom —— 体积稍大，界面美观

##### ( 前端开发需要依赖大量的 插件，插件，插件！！！ )
